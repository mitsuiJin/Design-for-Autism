# Design-for-Autism
산업디자인학과 졸업 프로젝트 지원

## 프로젝트 개요

‘Design for Autism’은 자폐 스펙트럼 장애(ASD) 아동의 **언어 표현력과 사회적 의사소통 능력 향상**을 목표로 한 감각 통합형 학습 도구 디자인 프로젝트입니다.

시각·촉각 자극을 결합한 피지컬 토이와 AR 애플리케이션을 연동해, 아동이 놀이를 통해 자연스럽게 언어와 사회적 행동을 학습하도록 돕는 것이 핵심입니다.

## 주요 목표

- 피지컬 토이와 디지털 콘텐츠를 연결해 **상호작용 기반 학습 경험 제공**
- 자폐 아동이 가정과 사회 환경 속 행동을 자연스럽게 학습하도록 지원

## 담당 역할 및 주요 기여

- **Unity 기반 AR 애플리케이션 전체 개발 담당**
- **Vuforia 플러그인 기반 마커 인식 시스템 구현**
- Vuforia가 **후면카메라**를 우선으로 인식하는 문제를 AR Session Origin의 **AR Face Manager 컴포넌트를 결합**함으로써 창의적으로 해결
- 프로젝트 발표 전까지 완성 가능한 AR 시스템 구축 및 최적화
